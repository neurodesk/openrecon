# neurorecon
repository for building openrecon containers

for running locally:

initial setup:
```
python3 -m venv .venv
```

run locally:
```
source .venv/bin/activate
<!-- cd recipes/THE_RECIPE_YOU_WANT_TO_BUILD e.g.:-->
cd recipes/musclemap
cd recipes/spinalcordtoolbox
/bin/bash ../build.sh
```
